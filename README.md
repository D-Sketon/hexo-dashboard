# hexo-dashboard

use React 18 to refactor [hexo-admin](https://github.com/jaredly/hexo-admin)  
still WIP

## Start

```bash
pnpm i
cd ./client
pnpm i
```

## Build

```bash
pnpm run build
cd ./client
pnpm run build
```
